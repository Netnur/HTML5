HTML5 Semantic and Deprecated Tags

New Semantic Tags

<header>

This tag represents the header of a page or section. It usually contains introductory content or navigational links.

<header>
    <h1>Website Title</h1>
    <nav>...</nav>
</header>

<nav>

This tag is used for navigation links.

<nav>
    <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About</a></li>
        <li><a href="#contact">Contact</a></li>
    </ul>
</nav>

<main>

This tag represents the dominant content of the <body> of a document.

<main>
    <h2>Main Content</h2>
    <p>This is the main content of the page.</p>
</main>

<article>

This tag represents a self-contained piece of content that could be distributed and reused independently.

<article>
    <h2>Article Title</h2>
    <p>This is the content of the article.</p>
</article>

<section>

This tag represents a generic section of a document. It is used to group related content.

<section>
    <h2>Section Title</h2>
    <p>This is the content of the section.</p>
</section>

<aside>

This tag represents a section that is tangentially related to the content around it.

<aside>
    <h3>Related Information</h3>
    <p>This is additional information related to the main content.</p>
</aside>

<footer>

This tag represents the footer of a page or section. It typically contains copyright information, contact details, or site maps.

<footer>
    <p>© 2024 Your Company</p>
    <a href="privacy.html">Privacy Policy</a>
</footer>

<figure>

This tag represents any content that is referenced from the main content, such as images or diagrams.

<figure>
    <img src="image.jpg" alt="Description of the image">
    <figcaption>Image caption goes here.</figcaption>
</figure>

<figcaption>

This tag provides a caption or description for a <figure> element.

<figcaption>Image caption goes here.</figcaption>

<time>

This tag represents a specific time or date.

<time datetime="2024-07-31">July 31, 2024</time>

Deprecated Tags

<font>

This tag was used to change font attributes. CSS is recommended for styling.

<font color="red">Red Text</font>

<center>

This tag was used to center-align text or elements. CSS is recommended for alignment.

<center>Centered Text</center>

<big>

This tag was used to make text larger. CSS is recommended for text size.

<big>Big Text</big>

<strike>

This tag was used to apply a strikethrough to text. CSS is recommended for text decoration.

<strike>Strikethrough Text</strike>

<u>

This tag was used to underline text. CSS is recommended for text decoration.

<u>Underlined Text</u>

<b>

This tag was used to make text bold. CSS is recommended for text weight.

<b>Bold Text</b>

<i>

This tag was used to italicize text. CSS is recommended for text style.

<i>Italic Text</i>
